Access Keys

List of Access Keys

To the right is a list of access keys which allow you to navigate through the webpage without using the mouse.

How To Use

To use the access keys, please follow instructions below:

  1. Hold down the "alt" key on the keyboard
  2. Press the corrisponding key on the keyboard to open the page
Access Key List
Home Page: alt+h
About: alt+a
Contact dropdown: alt+c
    Contact Information: alt+c then alt+1
    Contact Form: alt+c then alt+2
Social Events Dropdown: alt+s
    Social Events: alt+s then alt+3
    Club Holiday: alt+s then alt+4
FAQ + Information Dropdown: alt+i
    FAQ: alt+i then alt+5
    Club Sessions: alt+i then alt+6
    Our Racing Team: alt+i then alt+7
    List of Access Keys: alt+i then alt+8
